  快速開始（3 步驟）
═══════════════════════════════════════════════════════════════

1️⃣ 安裝套件
   
   npm install

2️⃣ 設定 Google Maps API Key（重要！）
   
   - 詳細步驟請參考：API_KEY_設定指南.md
   - 或快速參考：快速開始.md
   - 建立 .env 檔案並加入你的 API Key

3️⃣ 啟動開發伺服器
   
   npm run dev
   
   然後開啟瀏覽器前往：http://localhost:3000

═══════════════════════════════════════════════════════════════
  重要提醒
═══════════════════════════════════════════════════════════════

⚠️  Google Maps API Key 是必要的！
   沒有 API Key 地圖功能無法運作
   請參考 API_KEY_設定指南.md 進行設定

⚠️  需要建立 .env 檔案
   在專案根目錄建立 .env 檔案
   內容：VITE_GOOGLE_MAPS_API_KEY=你的API金鑰

⚠️  不要將 .env 提交到 Git
   API Key 應該保密
   .gitignore 已經設定好了

⚠️  資料儲存在 LocalStorage
   清除瀏覽器資料會刪除所有行程
   建議定期備份重要資料

═══════════════════════════════════════════════════════════════

📱 安裝到手機或桌面
   1. 使用 Chrome 或 Edge 開啟應用程式
   2. 點擊瀏覽器的「安裝」按鈕
   3. 應用程式會出現在主畫面或應用程式列表

💡 PWA 圖標
   • 需要準備 192x192 和 512x512 的圖標
   • 詳細說明請看 PWA圖標說明.md
   • 可以暫時使用預設圖標
